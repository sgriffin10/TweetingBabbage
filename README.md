# TweetingBabbage

We used all of these requirements including the tutorial pages from Raspberry Pi to teach us how to: 
- Write the code that programmed the Raspberry Pi and Camera to take picture
- Install the required packages for Raspberry Pi ( NOOBS )
- Format the Micro USB using an SD Formatter in order to empty its content
- Power on the Raspberry Pi by connecting: 
  - HDMI to Moniter (TV), 
  - USB Power adapter, 
  - Keyboard,
  - Mouse, and
  - Camera Module
- Boot the Raspberry Pi using LINUX
- Learn how to push changes of the code to Github for collaboration using LINUX command line 
- Take Pictures 
- How to upload to Twitter
